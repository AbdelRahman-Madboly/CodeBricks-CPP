## üìñ Project Overview
**CodeBricks** is a structured engineering portfolio documenting the transition from **C++ Fundamentals** to **Object-Oriented Programming (OOP)**, and finally to **Modern C++**.

This repository is not just code snippets; it is a roadmap designed to build the "Software Engineer Mindset" required for:
1.  **Embedded Systems** (Memory management, Bitwise operations)
2.  **Computer Vision & AI** (Data structures, Algorithmic efficiency)
